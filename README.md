# Medellín Joven - CMR

Sistema de gestión de Posibles Aliados para Medellín Joven.

## Características

- ✅ Gestión de posibles aliados
- ✅ Tabla con filtros y búsqueda
- ✅ Vista de detalle con información completa
- ✅ Modal para registrar nuevos aliados
- ✅ Timeline de actividades
- ✅ Dashboard con estadísticas

## Tecnologías

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Lucide React (iconos)

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Estructura del Proyecto

```
src/
├── components/      # Componentes reutilizables
├── pages/          # Páginas principales
├── types/          # Definiciones TypeScript
├── data/           # Datos mock
└── App.tsx         # Componente principal
```
